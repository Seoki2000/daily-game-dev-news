개발자 및 사용 가이드

이 페이지는 Daily Game Dev & AI Insights 시스템을 유지보수하고 커스터마이징하는 방법을 안내합니다.

1. 새로운 구독처(RSS/유튜브) 추가하기

새로운 블로그, 서브스택, 유튜브 채널의 글을 수집하려면 collector.py 파일을 텍스트 편집기(메모장, VS Code 등)로 열고 FEEDS 리스트에 주소를 추가하면 됩니다.

FEEDS = [
    # 기존 항목들...
    {"url": "https://추가할사이트주소/feed", "default_tag": "원하는태그명"},
    
    # 예시: 유튜브 채널 추가 방식 (채널 ID 필요)
    {"url": "https://www.youtube.com/feeds/videos.xml?channel_id=유튜브채널ID", "default_tag": "YouTube"}
]

2. 데이터 수집 자동화 (윈도우 스케줄러)

매일 수동으로 스크립트를 실행할 필요 없이, 컴퓨터가 켜져 있을 때 알아서 최신 글을 가져오도록 윈도우 스케줄러를 설정할 수 있습니다.

  • 윈도우 키를 누르고 "작업 스케줄러"를 검색하여 실행합니다.
  • 오른쪽 '작업 메뉴'에서 기본 작업 만들기를 클릭합니다.
  • 이름을 지정하고 (예: Daily News Update), 트리거를 '매일'로 설정합니다.
  • 작업은 '프로그램 시작'으로 선택합니다.
  • 프로그램/스크립트 항목에서 찾아보기를 눌러 프로젝트 폴더 안에 있는 run_update.bat 파일을 선택하고 마칩니다.

3. 작동 원리

  • collector.py: 등록된 주소에서 최신 글을 다운로드한 뒤, 설정된 Gemini API 키를 통해 글의 핵심을 3줄 요약하고 카테고리 태그를 추출하여 data/articles.js 파일에 저장합니다.
  • index.html / script.js: 웹 브라우저가 열릴 때 articles.js을 읽어와서 예쁜 카드로 렌더링하고 검색 기능을 제공합니다.
